1D Motion with Loops

Introduction

In this module, we will explore the basics of 1-dimensional motion and how to implement loops in Python to simulate motion. We will start with the equations of motion and then move on to using loops to calculate and simulate motion.

Equations of Motion

The equations of motion for an object in 1-dimensional motion are:

\begin{align*} x &= x_0 + v_0t + \frac{1}{2}at^2 \\ v &= v_0 + at \end{align*}

where x is the position, v is the velocity, a is the acceleration, and t is the time.

Loops in Python

Loops are a fundamental concept in programming that allow us to repeat a set of instructions multiple times. In Python, we can use for loops to simulate motion by updating the position and velocity at each time step. Here is an example of a loop that simulates motion:

for t in range(0, 10):
    x = x + v*t + 0.5*a*t**2
    v = v + a*t
    print("At time", t, "s, the position is", x, "m")

This loop calculates the position and velocity of an object at each time step, updates them using the equations of motion, and prints the position at each time step. We can use this loop to simulate motion in 1-dimensional space.
